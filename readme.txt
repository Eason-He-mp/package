================================================================================
                    CT 分块扫描自动化工具——使用说明
================================================================================

【文件清单】
  CT_Scan_Tool.exe    ->  主程序（自动化扫描）
  GetCoords.exe       ->  鼠标坐标抓取工具
  config.json         ->  配置文件（按钮坐标及窗口标题）

================================================================================
  一、初次使用：获取按钮坐标
================================================================================

  1. 在工业机上打开 CT 软件，并调整窗口至日常操作位置。
  2. 运行 GetCoords.exe，屏幕左上角会出现半透明坐标窗口。
  3. 鼠标依次悬停在软件界面的以下控件上，按 F12 复制坐标，
     或直接记录“Screen:”后面的数字：
        - X 坐标输入框
        - Y 坐标输入框
        - Live 按钮
        - Capture 按钮
        - 确认移动按钮
        - 打开保存对话框的按钮
        - 保存对话框中的“保存”按钮
  4. 用记事本打开 config.json，将上述坐标填入对应的字段。
     例如： "x_input": [450, 300]
     若不需要设置视野输入框，请将 size_input 设为 null。
  5. 按 Esc 退出坐标抓取工具。

================================================================================
  二、扫描准备
================================================================================

  1. 确保下列三个文件位于同一文件夹内：
        CT_Scan_Tool.exe
        GetCoords.exe
        config.json
  2. 打开 CT 软件，设置好电压、电流，并开启 X-Ray。
  3. 确认 CT 软件的窗口标题包含 config.json 中 window.title 字段
     所填写的文字，否则程序将无法找到窗口。

================================================================================
  三、执行扫描
================================================================================

  1. 点击 CT_Scan_Tool.exe，确保“以管理员身份运行”。
  2. 在弹出的界面中填写参数：
        X 轴起点 / X 轴终点 (mm)  ：起点应大于终点
        Y 轴起点 / Y 轴终点 (mm)  ：起点应小于终点
        视野大小 (mm)              ：单次扫描的正方形视野边长
        重叠比例 (0~1)            ：通常设为 0.15
        采集等待时间 (s)          ：捕获图像后等待的时间
        移动等待时间 (s)          ：CNC 移动到位所需的时间
        文件名前缀               ：保存图像的名称前缀，自动追加三位编号
  3. 点击“开始扫描”，在弹出的确认框中核对扫描块数，确认后开始。
  4. 程序将自动执行以下流程：
        Live -> 设置 X/Y -> 确认移动 -> 等待到位 -> Capture -> 保存 -> Live
  5. 扫描顺序：
        从左到右、从上到下（先固定 Y，X 递减扫描；一行完成后再换下一行）
  6. 扫描过程中可按键盘左上角的 Esc 键紧急停止，鼠标将自动移到屏幕中央。
  7. 在开始扫描前保存一次空白内容以固定保存路径
  8. 全部扫描结束后弹窗提示，所有图像保存在默认目录中。
