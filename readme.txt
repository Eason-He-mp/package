CT分块扫描自动化工具 使用说明
================================

【文件清单】
- CT_Scan_Tool.exe    → 主程序（自动化扫描）
- GetCoords.exe       → 鼠标坐标抓取工具
- config.json         → 配置文件（按钮坐标及窗口标题）

================================
一、初次使用：获取按钮坐标
================================
1. 在工业机上打开CT软件，并调整窗口至日常操作位置。
2. 运行 GetCoords.exe，屏幕左上角会出现半透明坐标窗口。
3. 鼠标依次悬停在软件界面的以下控件上，记下“Screen:”后的坐标（或按F12复制）：
   - X坐标输入框
   - Y坐标输入框
   - Live按钮
   - Capture按钮
   - 确认移动按钮
   - 打开保存对话框的按钮
   - 保存对话框中的文件名输入框
   - 保存对话框中的“保存”按钮
4. 用记事本打开 config.json，将上述坐标填入对应字段（若不需要视野输入框则设为 null）。
5. 按 Esc 退出抓取工具。

================================
二、扫描准备
================================
1. 将 CT_Scan_Tool.exe、GetCoords.exe、config.json 放在同一个文件夹内。
2. 打开CT软件，设置好所需的电压、电流，并确保X-Ray已开启。
3. 确认CT软件窗口标题包含 config.json 中 window.title 字段的文字（否则程序无法找到窗口）。

================================
三、执行扫描
================================
1. 右键点击 CT_Scan_Tool.exe → 以管理员身份运行。
2. 在弹出的界面中填写参数：
   - X轴起点 / X轴终点 (mm)：起点应大于终点（从左往右递减）
   - Y轴起点 / Y轴终点 (mm)：起点应小于终点（从下往上递增）
   - 视野大小 (mm)
   - 重叠比例 (0~1)：通常0.15
   - 采集等待时间 (s)：捕获图像后等待保存的时间
   - 移动等待时间 (s)：CNC移动到位所需时间
   - 文件名前缀：保存图像的前缀，自动追加三位编号
3.在扫描开始前保存一个空白图像以固定保存路径
4. 点击“开始扫描”，在弹出的确认框中核对扫描块数，确认后开始。
5. 程序将自动执行以下流程：
   Live → 设置X/Y → 确认移动 → 等待到位 → Capture → 保存 → Live
   扫描顺序：从左到右、从上到下（先固定Y，X递减；一行完成后再换下一行）。
6. 扫描过程中可按 Esc 键紧急停止，鼠标会自动移到屏幕中央。
7. 完成后弹窗提示，所有图像保存在默认目录中。
